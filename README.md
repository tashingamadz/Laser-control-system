# Laser-control-system
Laser system control system that has different channels and commands

# Features
Display different laser statuses for each channel
Dynamic channel system
GUI
Terminal based commands
Threading

# Technologies
Pyhton
JSON file
Raspberry Pi4

# How to run
1. Open git repository
2. Clone it
3. Copy URL
4. Open IDE (preferrably VS Code)
5. Paste URL in terminal
6. cd repository
7. Run the terminal

   Or
1. Open GitHub Desktop
2. Select your repository from the list on the left
3. At the top, click Repository
4. Open in Visual Studio Code
